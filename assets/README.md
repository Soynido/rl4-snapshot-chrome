# Assets

## Logo

Place the RL4 logo here:
- `assets/logo.png` (recommended: 512x512px or higher)
- `assets/logo.svg` (vector format, preferred)

The logo should be the abstract black design on light pink background (four interconnected rounded shapes with a circular element below).

## Usage

The logo is referenced in:
- README.md (header)
- GitHub repo social preview
- Extension icons (if needed)
